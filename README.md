
# AUTOSKP2GLB

AUTOSKP2GLB is a program designed to automatically convert SketchUp (SKP) files into GLB format and upload them to Amazon S3 for storage

## Software

| Software | Version  |
| :-------- | :------- |
| `SketchupSDK` | `24.0.484` |
| `C++ Build Tools` | `MSVC v143` |
